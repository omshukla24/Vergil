• conf < threshold
• Status: PENDING → APPROVED
• Auth0 MFA per-action URL
• RS256 JWT via JWKS

• Sensitive / high-risk
• ≥2 trustees required
• Redis WATCH/MULTI atomic
• Exponential backoff poll

• Quorum timeout fallback
• Override zero-trust lock
• Temp elevated privileges
• Auto-resets after use
